### Tipos de datos primitivos
- [String] - Contienen cadenas de caracteres (texto)
- [number] - tanto flotantes como enteros, no hay distinción
- [boolean] - Dedicados para operaciones lógicas
- [undefined] - Cuando una variable esta definida pero no inicializada
- [null] - Cuando una variable esta vacia intencionalmente
- [Symbol] - Similar a las primarykeys, representan un valor único e irrepetible
- [bigInt] - Numero que superan los 8bytes normales.

### Estructuras de datos
- [Objeto] - Elemento que define atributos y métodos
- [Array] - Arreglo de la memoria que almacena elementos de forma conjunta
- [Función] - Serie de instrucciones agrupadas
- [Fecha] - Tipo de dato que contiene fecha y hora
